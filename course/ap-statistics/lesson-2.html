Frequency Tables

What Is a Frequency Table?

A frequency table shows how often each value (or category) occurs in a data set. It's one of the simplest and most useful ways to organize data.

Building a Frequency Table

A frequency table typically has these columns:

  • Value — The category or data value
  • Frequency — How many times that value appears
  • Relative Frequency (optional) — The fraction or percentage of the total

Example

A survey asks 25 students their favorite fruit:

FruitFrequencyRelative Frequency
Apple1212/25 = 48%
Banana88/25 = 32%
Orange55/25 = 20%
Total25100%
Key Insight: Relative frequency is calculated as: \(\text{Relative Frequency} = \frac{\text{Frequency}}{\text{Total}}\). All relative frequencies must add up to 1 (or 100%).
