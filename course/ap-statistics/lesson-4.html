Stem-and-Leaf Plots

What Is a Stem-and-Leaf Plot?

A stem-and-leaf plot (or stemplot) splits each number into two parts: a stem (the leading digit or digits) and a leaf (the last digit). This lets you see the shape of the data while preserving every original value.

Example

Data: 23, 25, 31, 32, 38, 41, 45, 47

StemLeaf
23  5
31  2  8
41  5  7

Key: 2 | 3 = 23

Why Use Stem-and-Leaf Plots?

  • Shows the shape of the distribution (like a histogram turned sideways)
  • Preserves individual data values — you can read every number
  • Easy to find the median, mode, and range
  • Helps spot clusters, gaps, and outliers
Key Insight: A back-to-back stem-and-leaf plot uses the same stems with leaves extending in both directions, allowing you to compare two data sets side by side.
