Dot Plots

What Is a Dot Plot?

A dot plot (or line plot) displays data by placing a dot above a number line for each data value. If a value appears more than once, dots are stacked vertically.

Example

Test scores: 7, 8, 8, 9, 9, 9, 10

•
•       •
•   •   •   •
7   8   9   10

Reading a Dot Plot

  • Clusters — Groups of dots close together show where data concentrates
  • Gaps — Empty spaces with no dots show ranges with no data
  • Outliers — Dots far away from the rest of the data
  • Mode — The value with the tallest stack of dots (most frequent)

When to Use Dot Plots

Dot plots work best for small to medium data sets. They become cluttered with very large data sets (thousands of values). They are the simplest type of graph to create by hand.

Key Insight: Dot plots show every individual data point, making them great for spotting patterns, clusters, gaps, and outliers at a glance.
