Unit 1 — Limits

Continuity

A function \(f\) is continuous at \(x = a\) if and only if all three of the following conditions are met:

  • 1. \(f(a)\) is defined (the function has a value at \(a\)).
  • 2. \(\displaystyle\lim_{x \to a} f(x)\) exists (the limit from both sides agrees).
  • 3. \(\displaystyle\lim_{x \to a} f(x) = f(a)\) (the limit equals the function value).

If any of these three conditions fails, the function is discontinuous at \(x = a\).

Example 1: Is \(f(x) = x^2\) continuous at \(x = 3\)?

  • \(f(3) = 9\) — defined. ✓
  • \(\lim_{x \to 3} x^2 = 9\) — exists. ✓
  • \(\lim_{x \to 3} x^2 = f(3) = 9\) — equal. ✓

All three conditions hold, so \(f\) is continuous at \(x = 3\).

Example 2: Let \(g(x) = \dfrac{x^2 - 1}{x - 1}\). Is \(g\) continuous at \(x = 1\)?

  • \(g(1) = \frac{0}{0}\) — undefined. ✗

Condition 1 fails immediately. The function has a removable discontinuity (hole) at \(x = 1\), even though the limit exists and equals 2.

Key facts: Polynomials are continuous everywhere. Rational functions are continuous wherever the denominator is nonzero. Trig functions like \(\sin x\) and \(\cos x\) are continuous on all of \(\mathbb{R}\).
