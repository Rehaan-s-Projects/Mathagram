Unit 5 — Applications of Derivatives I

Derivative at a Value

The derivative at a specific value \(f'(a)\) is a single number — the instantaneous rate of change of \(f\) at the point \(x = a\).

$$f'(a) = \lim_{h\to 0} \frac{f(a+h) - f(a)}{h}.$$

But after you know the differentiation rules, the easier path is:

  • Find the general derivative \(f'(x)\) using the rules.
  • Plug in \(x = a\).

Example 1. For \(f(x) = x^2 - 3x\), find \(f'(4)\).

\(f'(x) = 2x - 3\), so \(f'(4) = 5\).

Example 2. For \(f(x) = \dfrac{1}{x}\), find \(f'(2)\).

\(f'(x) = -x^{-2}\), so \(f'(2) = -\dfrac{1}{4}\).

Example 3 — trig. For \(f(x) = \sin x\), find \(f'(\pi)\).

\(f'(x) = \cos x\), so \(f'(\pi) = -1\).

Interpretations.

  • Slope of the tangent line to \(y = f(x)\) at \(x = a\).
  • Instantaneous rate of change — how fast \(f\) changes at that exact input.
  • If \(f(t)\) is position, \(f'(a)\) is velocity at time \(a\).

Alternative form (useful on AP problems):

$$f'(a) = \lim_{x\to a}\frac{f(x) - f(a)}{x - a}.$$
