Unit 1 — Limits

Limits by Direct Evaluation

A limit describes what value a function approaches as its input approaches some number. We write:

$$\lim_{x \to a} f(x) = L$$

This reads: "the limit of \(f(x)\) as \(x\) approaches \(a\) equals \(L\)."

The simplest way to find a limit is direct substitution — just plug in the value of \(a\). If the function is continuous at \(a\) (no holes, no jumps, no vertical asymptotes), then:

$$\lim_{x \to a} f(x) = f(a)$$

Example 1. Find \(\lim_{x \to 3}(2x + 1)\).

Plug in \(x = 3\): \(2(3) + 1 = 7\). So the limit is \(7\).

Example 2. Find \(\lim_{x \to 4} \sqrt{x}\).

Plug in: \(\sqrt{4} = 2\). The limit is \(2\).

Example 3. Find \(\lim_{x \to 0} \cos x\).

Plug in: \(\cos(0) = 1\). The limit is \(1\).

When does direct substitution work? It works for polynomials, rational functions (when the denominator isn't zero), trig functions, exponentials, and logarithms — anywhere the function is continuous at the point.

When does it fail? When plugging in gives \(\frac{0}{0}\), \(\frac{\infty}{\infty}\), or an undefined expression. In those cases, you need other techniques (factoring, conjugates, L'Hôpital's Rule) which come in later lessons.
