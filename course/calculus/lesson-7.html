Unit 1 — Limits

Continuity

Intuitively, a function is continuous if you can draw its graph without lifting your pen. But in calculus we need a precise definition.

A function \(f\) is continuous at \(x = a\) if all three conditions hold:

  • 1. \(f(a)\) is defined (the point exists).
  • 2. \(\displaystyle\lim_{x \to a} f(x)\) exists (left and right limits agree).
  • 3. \(\displaystyle\lim_{x \to a} f(x) = f(a)\) (the limit equals the function value).

If any one of these fails, we have a discontinuity. There are three main types:

  • Removable discontinuity (hole): The limit exists but \(f(a)\) is either missing or doesn't match. You could "fill in" the hole to fix it.
  • Jump discontinuity: The left-hand and right-hand limits exist but are different. Think of a step function.
  • Infinite discontinuity: The function blows up to \(\pm\infty\) near \(x = a\). This creates a vertical asymptote.

Example: Consider \(\displaystyle f(x) = \frac{x^2 - 1}{x - 1}\). At \(x = 1\), the function is undefined (division by zero). But:

$$\lim_{x \to 1} \frac{x^2 - 1}{x - 1} = \lim_{x \to 1} \frac{(x-1)(x+1)}{x-1} = \lim_{x \to 1} (x+1) = 2$$

The limit exists and equals 2, but \(f(1)\) is undefined. This is a removable discontinuity.

Key fact: Polynomials, exponentials, sine, and cosine are continuous everywhere. Rational functions are continuous wherever the denominator is nonzero.
