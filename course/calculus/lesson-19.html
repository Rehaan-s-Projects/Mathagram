Unit 1 — Limits

Epsilon-Delta Definition

So far we have worked with limits informally — "\(f(x)\) gets close to \(L\) as \(x\) gets close to \(a\)." The epsilon-delta (\(\varepsilon\)-\(\delta\)) definition makes this precise.

Formal definition: We write \(\lim_{x \to a} f(x) = L\) if

$$\forall\, \varepsilon > 0,\ \exists\, \delta > 0 \ \text{such that}\ 0 < |x - a| < \delta \ \Longrightarrow\ |f(x) - L| < \varepsilon.$$

The story: No matter how tight a tolerance \(\varepsilon\) someone demands around \(L\), we can always respond with a tolerance \(\delta\) around \(a\) such that every \(x\) within \(\delta\) of \(a\) (except possibly \(a\) itself) lands within \(\varepsilon\) of \(L\).

Why \(0 < |x - a|\)? The condition \(0 < |x - a|\) excludes \(x = a\). Limits describe behavior near \(a\), not the value at \(a\) itself.

Translating the symbols:

  • \(|x - a| < \delta\) means \(x\) is inside the interval \((a - \delta,\ a + \delta)\).
  • \(|f(x) - L| < \varepsilon\) means \(f(x)\) is inside \((L - \varepsilon,\ L + \varepsilon)\).

Example — a linear limit. To prove \(\lim_{x \to 2}(3x + 1) = 7\): given any \(\varepsilon > 0\), we need \(|3x + 1 - 7| < \varepsilon\), i.e. \(3|x - 2| < \varepsilon\), so choosing \(\delta = \varepsilon/3\) works. Whenever \(0 < |x - 2| < \delta\), we get \(|3x + 1 - 7| < \varepsilon\).

Visualization. Picture a horizontal band of width \(2\varepsilon\) centered on \(L\), and a vertical band of width \(2\delta\) centered on \(a\). The limit exists if we can always make the vertical band narrow enough to force the graph into the horizontal band.
