Unit 1 — Limits

Formal Limit Proofs

Once you understand the \(\varepsilon\)-\(\delta\) definition, the next step is writing clean proofs that specific limits hold.

The proof template. To prove \(\lim_{x \to a} f(x) = L\):

  • Let \(\varepsilon > 0\) be arbitrary.
  • Scratch work: start from \(|f(x) - L| < \varepsilon\), manipulate to get it in terms of \(|x - a|\), and read off a value of \(\delta\).
  • Formal proof: choose that \(\delta\) and verify that \(0 < |x - a| < \delta\) implies \(|f(x) - L| < \varepsilon\).

Example 1 — linear. Prove \(\lim_{x \to 2}(3x - 1) = 5\).

Scratch: \(|(3x - 1) - 5| = |3x - 6| = 3|x - 2|\). We want this \(< \varepsilon\), so \(|x - 2| < \varepsilon/3\). Choose \(\delta = \varepsilon/3\).

Proof: let \(\varepsilon > 0\) and set \(\delta = \varepsilon/3\). If \(0 < |x - 2| < \delta\), then \(|(3x - 1) - 5| = 3|x - 2| < 3\delta = \varepsilon.\ \Box\)

Example 2 — quadratic. Prove \(\lim_{x \to 3} x^2 = 9\).

Scratch: \(|x^2 - 9| = |x - 3||x + 3|\). The troublesome factor is \(|x + 3|\). Bound it by first restricting to \(|x - 3| < 1\): then \(2 < x < 4\), so \(5 < x + 3 < 7\), giving \(|x + 3| < 7\). So \(|x^2 - 9| < 7|x - 3|\). For this to be \(< \varepsilon\), take \(|x - 3| < \varepsilon/7\). Combining: \(\delta = \min(1,\ \varepsilon/7)\).

Common trick: for nonlinear \(f\), first restrict \(\delta \le 1\) to bound the "extra" factor, then set \(\delta = \min(1, \varepsilon/M)\) where \(M\) is that bound.

Example 3 — constant. Prove \(\lim_{x \to a} c = c\). For any \(\varepsilon > 0\), any \(\delta > 0\) works, since \(|c - c| = 0 < \varepsilon\) always.
