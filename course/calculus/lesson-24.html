Differentiation Rules with Tables

Many AP-style problems give you a table of values for \(f,\ g,\ f',\ g'\) at specific inputs, then ask you to compute a derivative at a point without ever knowing the explicit formulas. This tests whether you know the rules, not the algebra.

Reference table (used in the examples below):

\(x\)\(f(x)\)\(f'(x)\)\(g(x)\)\(g'(x)\)
1234-1
25012
47-263

Example 1 — product rule. Find \((fg)'(1)\).

$$(fg)'(1) = f'(1)g(1) + f(1)g'(1) = 3\cdot 4 + 2\cdot(-1) = 10.$$

Example 2 — quotient rule. Find \(\left(\dfrac{f}{g}\right)'(2)\).

$$\frac{f'(2)g(2) - f(2)g'(2)}{g(2)^2} = \frac{0\cdot 1 - 5\cdot 2}{1^2} = -10.$$

Example 3 — chain rule. Let \(h(x) = f(g(x))\). Find \(h'(1)\).

$$h'(1) = f'(g(1))\cdot g'(1) = f'(4)\cdot(-1) = (-2)(-1) = 2.$$

Tips.

  • Write the rule first, then plug in numbers.
  • When you see a composition, the chain rule multiplies — never add.
  • If a needed value isn't in the table, re-check your setup.
