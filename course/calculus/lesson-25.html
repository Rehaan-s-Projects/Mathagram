Unit 4 — Implicit & Inverse

Implicit Differentiation

Some curves can't be written as \(y = f(x)\) — for example, the circle \(x^2 + y^2 = 25\). You can still find \(\dfrac{dy}{dx}\) using implicit differentiation: differentiate both sides with respect to \(x\), treating \(y\) as an unknown function of \(x\), then solve for \(\dfrac{dy}{dx}\).

The one rule to remember. Every time you differentiate a \(y\)-term, the chain rule tacks on a \(\dfrac{dy}{dx}\):

$$\frac{d}{dx}[\,y^2\,] = 2y\cdot\frac{dy}{dx},\qquad \frac{d}{dx}[\,y\,] = \frac{dy}{dx}.$$

Example 1 — the circle. Differentiate \(x^2 + y^2 = 25\).

$$2x + 2y\frac{dy}{dx} = 0 \ \Longrightarrow\ \frac{dy}{dx} = -\frac{x}{y}.$$

Example 2 — a product. Find \(\dfrac{dy}{dx}\) for \(x^2 y + y^3 = 10\).

Differentiate term by term, using the product rule on \(x^2 y\):

$$2xy + x^2\frac{dy}{dx} + 3y^2\frac{dy}{dx} = 0.$$

Collect the \(\dfrac{dy}{dx}\) terms and solve:

$$\frac{dy}{dx} = -\frac{2xy}{x^2 + 3y^2}.$$

Example 3 — a point. For the circle above, find the slope at \((3, 4)\): \(\dfrac{dy}{dx} = -\dfrac{3}{4}\).

Procedure.

  • Differentiate each side with respect to \(x\).
  • Whenever you differentiate a \(y\)-term, multiply by \(\dfrac{dy}{dx}\).
  • Algebraically solve for \(\dfrac{dy}{dx}\).
  • If asked for a slope, plug in the point at the end.
