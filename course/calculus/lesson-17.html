Unit 1 — Limits

Horizontal Asymptotes

A horizontal asymptote is a horizontal line that the graph of a function approaches as \(x \to \infty\) or \(x \to -\infty\).

$$\text{If } \lim_{x\to\infty} f(x) = L \text{ or } \lim_{x\to-\infty} f(x) = L,\ \text{then } y = L \text{ is a horizontal asymptote.}$$

A function can have at most two horizontal asymptotes — one as \(x \to \infty\) and one as \(x \to -\infty\). Often they are the same line.

Rational functions — degree rules: For \(f(x) = \dfrac{p(x)}{q(x)}\) with \(\deg p = m\) and \(\deg q = n\):

  • If \(m < n\): HA is \(y = 0\).
  • If \(m = n\): HA is \(y = \dfrac{a_m}{b_n}\) (ratio of leading coefficients).
  • If \(m > n\): no horizontal asymptote.

Example 1: \(f(x) = \dfrac{3x^2 + 1}{x^2 - 4}\). Degrees match, leading coefficients are 3 and 1, so \(y = 3\) is the horizontal asymptote.

Example 2: \(f(x) = \dfrac{5x}{x^2 + 7}\). Numerator degree is smaller, so \(y = 0\) is the horizontal asymptote.

Example 3: \(f(x) = \dfrac{x^3 + 2}{x^2 - 1}\). Numerator degree is larger, so there is no horizontal asymptote (instead, the function grows without bound).

Example 4 — exponential: \(\lim_{x\to\infty} e^{-x} = 0\), so \(y = 0\) is a horizontal asymptote of \(e^{-x}\).
