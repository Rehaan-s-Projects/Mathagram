Water Sort — How It Works

The Goal

Water Sort is a stacking puzzle. You start with a row of tubes; each tube holds several layers of colored liquid stacked on top of one another. Some tubes are full of mixed colors; one or two start empty. You win when every tube either is empty or contains only one color.

The One Pour Rule

You can only pour from one tube to another when:

  • The destination tube is empty, OR
  • The top layer of the destination matches the top layer of the source.

When you pour, every connected layer of the matching color moves at once — not just one unit. So if the source has three reds on top, all three flow into the destination (if there's room).

Core Strategies

  • Free up empty tubes early. An empty tube is a wildcard — it can accept any color first. Don't waste it on something you can place elsewhere.
  • Consolidate matching tops. If two tubes both show red on top, pour the smaller stack onto the larger to free up a tube faster.
  • Look ahead. A pour that solves now can trap a color underneath later. Trace what each color needs before committing.
  • Reset when stuck. If every tube has mixed colors with no empty space and no matching tops anywhere, the level isn't solvable from this state — restart and try a different opening sequence.
Key insight: The hardest moves aren't the obvious matches — they're the moves that create matches you couldn't see yet. Look for the chain, not the single pour.
